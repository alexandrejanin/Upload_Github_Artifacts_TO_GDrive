# Support

## 🆘 How to get help

If you are having trouble with this GitHub Action, here are some ways to get help:

1.  **Check the Documentation**: Read the `README.md` and check the inputs table carefully.
2.  **Search Issues**: Look through [existing issues](https://github.com/Jumbo810/Upload_Github_Artifacts_TO_GDrive/issues) to see if someone else has had the same problem.
3.  **Open an Issue**: If you found a bug or have a feature request, [open a new issue](https://github.com/Jumbo810/Upload_Github_Artifacts_TO_GDrive/issues/new).

## 🐛 Reporting Bugs

When reporting a bug, please include:
- A clear description of the issue.
- The version of the action you are using (e.g., `v2.3.3`).
- Steps to reproduce the problem.
- Relevant logs (please mask any sensitive tokens).

Thank you for using our action!
